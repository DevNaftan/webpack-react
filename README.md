# Webpack config for React

A basic Webpack configuration for React proyects.

### Run project

```
cd webpack-react
npm i
npm start
```

### Build project

```
npm run build
```

### Loaders and plugins installed

- webpack
- webpack-cli
- webpack-dev-server
- @babel/core
- @babel/preset-env
- @babel/preset-react
- babel-loader
- css-loader
- css-minimizer-webpack-plugin
- mini-css-extract-plugin
- sass
- sass-loader
- html-loader
- html-webpack-plugin
- clean-webpack-plugin
